# **App Name**: JSONify

## Core Features:

- JSON Input: Text input area for raw JSON.
- Format JSON: Pretty print JSON for readability.
- Validate JSON: Check JSON syntax for errors and display messages.
- Minify JSON: Compress JSON to reduce size.
- AI-Powered Error Explanation: Use AI to explain JSON syntax errors and how to fix them using a tool.
- Smart Auto-Correction: Offer one-click fixes for common JSON issues with a preview before applying.
- Live Collaborative Mode: Real-time JSON editing with teammates, including comments and change history.
- Visual JSON Tree Explorer: Toggle between raw JSON and a collapsible tree view with icons and color-coded nesting.
- Theme Customizer with Presets: Let users choose from themes with accessibility presets.
- JSON-to-UI Preview: Render JSON as a mock UI to visualize structure.
- Schema-Aware Validation: Support JSON Schema and OpenAPI validation, highlighting mismatches.
- Multi-Format Converter: Convert JSON ↔ YAML ↔ XML ↔ TOML with syntax highlighting and clipboard/download options.
- Secure Mode: Redact sensitive keys before sharing and warn users if secrets are detected.
- Search & Filter: Search by key name, value, or type and filter to show specific paths.
- Performance Analyzer: Benchmark large JSON files and suggest optimizations.
- Offline Mode + PWA Support: Installable as a Progressive Web App with full offline functionality.

## Style Guidelines:

- Dark background for a sleek developer look: #121212
- Primary color: Electric blue (#7DF9FF) for highlighting interactive elements and keywords.
- Accent color: A lighter shade of electric blue (#A0FFFF) for buttons and active states.
- Font: 'Inter' for a modern, neutral sans-serif feel; used for both the UI and for formatted/validated code.
- Code Font: 'Source Code Pro' to ensure consistent width of characters when displaying formatted or validated code.
- Simple, outline-style icons to represent actions (format, validate, minify).
- Clean, spacious layout to maximize readability.
